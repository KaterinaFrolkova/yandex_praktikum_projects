# Сравнение тарифов мобильного оператора

**Данные**
В нашем распоряжении данные 500 пользователей мобильного оператора:
 * тарифы
 * статистика звонков
 * статистика сообщений
 
**Задачи**
определить какой тариф приносит больший доход

**Используемые библиотеки**

 * *pandas*
 * *numpy*
 * *seaborn*
 * *scipy.stats*
